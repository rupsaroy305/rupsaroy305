## 💼 Experience

### 🌿 Data Science Intern — Climate Resilience Observatory (CRO)
*Machine Learning Project: District-Level Daily Rainfall Prediction*

- Engineered and merged **234,000+ rows** of daily weather data from ERA5 (ECMWF) and NASA POWER across **72 districts of Uttar Pradesh** (2017–2025)
- Built a **dual-task ML pipeline**: XGBoost Classifier for rain/no-rain prediction and an XGBoost + LightGBM ensemble for rainfall amount (mm/day)
- Designed ~30 physical features — thermodynamic, temporal, wind, spatial, and lag variables
- Achieved **ROC-AUC of 0.963**, **88.6% test accuracy**, and **94% rain day recall** on the classification task
- Applied SMOTE oversampling, RobustScaler, and log1p target transformation to handle class imbalance and skewed distributions

## 🚀 Projects

### 🌧️ District-Level Rainfall Prediction — UP (72 Districts)
*Internship Project @ Climate Resilience Observatory*

End-to-end ML pipeline predicting daily rainfall for all 72 Uttar Pradesh districts using ERA5 + NASA POWER satellite data. Dual-task system: Rain/No-Rain classification and mm/day regression amount. Temporal train/val/test split used to prevent data leakage and mirror real forecasting conditions.

`Python` `XGBoost` `LightGBM` `SMOTE` `Pandas` `ERA5` `NASA POWER`

| Metric | Score |
|---|---|
| Test ROC-AUC | **0.963** |
| Test Accuracy | **88.6%** |
| Rain Day Recall | **94%** |
| Regression R² | **0.63** |
| Regression RMSE | **4.17 mm** |
